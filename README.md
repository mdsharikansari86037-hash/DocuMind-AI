# DocuMind AI

An AI-powered document analyzer built with FastAPI and Google Gemini API.

## Features

- Upload documents
- AI-powered document analysis
- Chat with uploaded documents
- FastAPI backend
- HTML, CSS, JavaScript frontend
- Docker support

## Tech Stack

- Python
- FastAPI
- Google Gemini API
- HTML
- CSS
- JavaScript
- Docker
